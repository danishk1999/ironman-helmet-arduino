# 🦾 Iron Man Helmet — Arduino Automation
**Course:** CMPT 312 — Intro to Robotics  
**Institution:** MacEwan University  
**Status:** ✅ Completed

## Overview
Designed and built a 3D printed Iron Man helmet with an automated faceplate 
that opens and closes using a hand wave gesture. The faceplate is controlled 
by an Arduino Uno paired with an ultrasonic distance sensor that detects hand 
proximity and triggers the servo motors accordingly.

## Hardware Components
| Component | Purpose |
|---|---|
| Arduino Uno | Microcontroller — reads sensor & controls servos |
| Ultrasonic Distance Sensor | Detects hand wave gesture for open/close trigger |
| Servo Motors | Physically opens and closes the faceplate |
| 3D Printed Parts | Helmet shell and faceplate components |

## How It Works
Hand Wave Detected
↓
Distance Sensor sends signal
↓
Arduino Uno processes input
↓
Servo Motors activate
↓
Faceplate Opens / Closes

## Key Features
- **Gesture-based control** — faceplate triggered by hand wave, no buttons needed
- **3D printed construction** — custom designed and printed helmet parts
- **Automated servo control** — smooth open and close motion
- **Distance threshold logic** — faceplate responds only within a set range

## Technologies Used
- Arduino Uno (C/C++)
- Ultrasonic Distance Sensor (HC-SR04)
- Servo Motor Control
- 3D Printing (PLA filament)

## What I Learned
- Arduino programming and hardware interfacing
- Sensor integration and threshold-based logic
- Servo motor control and calibration
- 3D design and printing for functional mechanical parts
- Combining electronics with physical fabrication
